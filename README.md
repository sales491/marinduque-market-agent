# Marinduque Market Analysis Agent

This repository contains the code and documentation for a localized Multi-Agent System designed to analyze the digital landscape of businesses in Marinduque, Philippines.

## Architecture

The system is composed of four primary sub-agents:

1. **Data Harvester**: Extracts unstructured raw data from local directories, Google Maps, and Facebook groups.
2. **Profile Synthesizer**: Cleans, deduplicates, and structures the data into rich business profiles with a Digital Maturity Score.
3. **Sentiment Analyst**: Analyzes customer feedback and community posts to uncover market trends and pain points.
4. **Agency Strategist**: Generates actionable intelligence reports and client pitch recommendations for the digital marketing agency.

See `docs/multi_agent_workflow.md` for detailed interaction patterns between these agents.
